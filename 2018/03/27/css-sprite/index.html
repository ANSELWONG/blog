    本post主要记录我开发中css sprite的构建过程

    首先开始用到了gulp和gulp-spritesmith，gulp工具和包的使用教程可以从官方的详细文档中得知。

    配置

    参见gulp.spritesmith官方文档

    配置好之后，在test目录下执行命令gulp sprite可以得到 public 目录

    以上我们根据 官方的默认的retina模板 得到相应的css和image，这时候我们把构建完成的目录根据需求放到正在开发的项目中相应能引用到的地方，就可以使用了。

    这个过程从整个img目录里面讲所有递归目录的.png文件打包成一张image，显然在实际开发过程这种做法只能满足短小的需求。
